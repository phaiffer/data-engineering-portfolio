# Docs

This folder keeps the project documentation close to the implementation.

## Guide

- `source.md`: source choice, bounded execution, and field notes.
- `architecture.md`: local event-driven architecture and responsibility boundaries.
- `bronze.md`: raw landing design and Bronze metadata layout.
- `silver.md`: standardization strategy and partitioned event outputs.
- `gold.md`: Gold table grain, metric definitions, and rebuild-oriented output layout.
- `checkpointing.md`: readable checkpoint artifacts and stop or resume behavior.
- `replay.md`: broker replay versus offline replay from landed Bronze files.
- `local-run.md`: practical commands for local execution.
- `validation.md`: what to inspect after a run and how to build the local validation manifest.
